Audio to MIDI

Drop an MP3 and get MIDI back.

A single-file Vercel-friendly transcription page with a Songscription-style flow: upload, transcribe, preview, and download. It uses a browser-side transcription attempt first, then builds a MIDI file right in the page.

Single HTML file Drag and drop Progress + preview MIDI download

Works best with

Single-instrument audio, especially piano, guitar, bass, voice, or isolated stems.

Files supported

MP3, OGG, WAV, and FLAC are the safest choices in the browser.

Important

This is a front-end build. For multi-instrument cloud-grade separation like Songscription, you would normally add a backend later.
